#### 🚰 [Drink Water App v2.1.0](https://play.google.com/store/apps/details?id=com.eahm.drinkwaterapp)
*Hydration reminders & habit tracking application*

`Kotlin` • `Jetpack Compose` • `Room` • `WorkManager` • `AdMob` • `RevenueCat`
</div>

- **Clean Architecture**: Native Kotlin app structured around Clean Architecture with reactive state management (restructured internally to a KMP project).
- **Background Jobs**: Integration of WorkManager for periodic background backups to Google Drive and local reminders.
- **Monetization**: Complete setup of in-app subscriptions and features via RevenueCat, alongside AdMob ad network integration.
- **Quality Control**: Automated pipelines with GitHub Actions, Renovate dependency updates, and Detekt code analysis.

<div align="center">

#### 💳 [Credit Manager v1.0.0](https://elvishrdz.github.io/#projects)
*Cross-platform client credits and accounting dashboard*

`Kotlin Multiplatform (KMP)` • `Compose Multiplatform (CMP)` • `Spring Boot` • `PostgreSQL` • `Room DB` • `GitHub Actions`
</div>

- **Shared Business Logic**: Core business logic centralized using KMP, integrated into the iOS side using Xcode-based workflows.
- **Unified UI**: User interface built with Compose Multiplatform (CMP) to deliver a cohesive, native visual experience across Android and iOS.
- **Offline-First**: Local persistence with Room Database optimized for Compose recomposition and low memory footprint.
- **Robust Backend**: Developed with Spring Boot and Kotlin, incorporating secure authentication rules, roles, and permissions.
